三、Slider滑动条组件

（1）属性和方法
maximumValue number
滑块的初始最大值。默认值为1。
minimumValue number
滑块的初始最小值。默认值为0。
onSlidingComplete函数
当用户完成更改值时调用回调（例如，释放滑块时）。
onValueChange函数
当用户拖动滑块时，连续调用回调。
滑块的步长值。该值应介于0和（maximumValue - minimumValue）之间。默认值为0。
